Hardware

This page will tell you about all of the types of
hardware in a computer, what they are called,
and their purposes.






Motherboard

The motherboard is a sheet of plastic that holds all
the circuitry to connect the various components of a computer
system. Learn how the motherboard functions to make all the
other components work together.


CPU

The central processing unit (CPU) is the brain of your
computer. It handles all the instructions you give your
computer, and the faster it does this, the better. Learn
about how a CPU processes instructions and how computer
engineers are continuously coming up with ways to make it
go faster.


RAM

Random-access memory (RAM) is a type of storage for
computer systems that makes it possible to access data very
quickly in random order. The term RAM has become
associated with the main memory of a computer system.


Hard drive

There are several different types of hard drives for storing
software and data files: hard disk drives, solid-state drives and
external drives. Learn about how they work and how to select
the right drive for a particular computer system in this lesson.
